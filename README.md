# tonyvazz.com

Personal site for Tony Vazz — marketing leader, founder of [Tecnato](https://tecnato.ai), and author of [Dados Sangrientos](https://dadossangrientos.com).

## Stack

- Single-page static site (`index.html`) with all CSS and JS inline.
- Google Fonts: DM Sans + Instrument Serif.
- No build step, no dependencies.
- Hosted on Vercel, deployed automatically on every push to `main`.

## Files

- `index.html` — the site
- `tony-vazz.jpg` — hero photo
- `tony-pfp.jpg` — round avatar in the quote section
- `about.html` — long-form bio (coming soon)

## Editing

Small text edits can be made directly in the GitHub web editor. Vercel picks up changes within ~30 seconds and redeploys.

For larger changes, edit `index.html` locally, then upload via **Add file → Upload files** in GitHub.

## Design system

- Background `#FAFAF8`
- Text `#0A0A0A`
- Accent (red) `#8B0000`
- Secondary text `#555550`
- Borders `#E8E8E4`

Matches [tecnato.ai](https://tecnato.ai) for visual consistency across the brand.
